# Typing Speed Test

Simple typing speed test web app.

How to run locally

1. Open `index.html` in your browser. If ES modules fail to load, run a local server:

```bash
python -m http.server 8000
# or
npx http-server .
```

This repository contains the static site files: `index.html`, `css/`, and `js/`.
